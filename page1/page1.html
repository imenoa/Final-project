l'HTML ou le squelette :

HyperText Markup Language

HTML est un langage utilisé pour structurer les pages web, un peu comme le squelette d'un site internet. Il permet d'organiser le contenu, comme les titres, les textes, les images, les liens, et les tableaux, afin que les navigateurs ( Google Chrome ou Firefox) sachent comment les afficher correctement. Chaque élément du site est entouré de balises (des mots entre "<" et ">" ) qui indiquent au navigateur ce qu'il doit faire.
Par exemple:
- Balise de titre (h1 à h6) utilisées pour structurer le contenu et hiérarchiser les informations, h1 permettand de créer un grand titre; - Balise p utilisée pour définir un paragraphe de texte.;
Le développeur web conçoit et entretient des sites et applications web en utilisant diverses technologies. Son travail est divisé en deux branches principales : le front-end, qui concerne l'interface visible par l'utilisateur (avec des langages comme HTML, CSS, et JavaScript), et le back-end, qui gère la logique du serveur et les bases de données (via des langages comme PHP, Python ou Node.js). Un développeur peut aussi être full-stack, c'est-à-dire qu'il maîtrise à la fois le front-end et le back-end. En plus, il doit s'assurer que les sites sont bien optimisés pour tous les appareils (ordinateurs, smartphones) et qu'ils respectent les règles du SEO pour être bien référencés sur les moteurs de recherche.
